# 三江源动物性格测试

基于大五人格理论 (Big Five / OCEAN) 的高原动物性格测试网站。

## 🌟 特性

- **36道专业测评题目** - 基于大五人格理论 + 高原情境投射题
- **5种高原动物图腾** - 雪豹、棕头鸥、牦牛、高山兀鹫、赤狐
- **人格雷达图** - 可视化展示你的五维人格得分
- **响应式设计** - 完美适配手机、平板、电脑
- **纯静态部署** - 无需后端，Vercel 免费托管

## 🚀 部署到 Vercel

### 方法一：Git 集成部署（推荐）

1. 将项目推送到 GitHub/GitLab/Bitbucket
2. 访问 [Vercel](https://vercel.com) 并登录
3. 点击 "Add New Project"
4. 选择你的仓库
5. 点击 "Deploy"

### 方法二：Vercel CLI

```bash
# 安装 Vercel CLI
npm i -g vercel

# 登录
vercel login

# 部署
vercel
```

### 方法三：拖拽部署

1. 访问 [Vercel](https://vercel.com)
2. 将项目文件夹拖拽到指定区域
3. 自动部署完成

## 📊 测评维度

基于大五人格模型 (OCEAN)：
- **Openness (开放性)** - 探索与创新
- **Conscientiousness (责任感)** - 计划与自律
- **Extraversion (外向性)** - 社交与活力
- **Agreeableness (宜人性)** - 合作与同理心
- **Neuroticism (情绪稳定性)** - 压力应对
